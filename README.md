## Features

This package is develop to speed up the process of UI development

## Get Started
You can start by changing your some widget to Az widgets like this
<br />
Row > AzRow
<br />
Column > AzColumn

### AzButton
Example of calling button
```
AzButton('Text')
.widthFull()
.primaryOutline()
.onPressed(() {
    print('On Pressed call this');
}),
```
